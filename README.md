# Professor Assistant




### Description

Professor Assistant is a Python application that helps professors create exams from a question bank.

The program reads a text file containing question-answer pairs, randomly shuffles the questions, selects the requested number of questions, and saves them into a new exam file.

The project demonstrates the use of modular programming, functions, file processing, exception handling, input validation, and Python's random library.
